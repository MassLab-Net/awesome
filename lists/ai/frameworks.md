# Frameworks

Resources about general AI frameworks.

## Semantic Kernel
- Link: https://github.com/microsoft/semantic-kernel
- Type: Framework / Official repo
- Why it matters: A major reference point for building AI apps and agents, especially when comparing approaches in the .NET ecosystem.
- Compare with: LangChain, LlamaIndex
- Notes: High-value reference for the `.NET + AI` track.

## LangChain
- Link: https://github.com/langchain-ai/langchain
- Type: Framework / Official repo
- Why it matters: A widely used reference point for LLM workflows, agents, and integrations.
- Compare with: Semantic Kernel, LlamaIndex
- Notes: Often used as a baseline when evaluating framework tradeoffs.
