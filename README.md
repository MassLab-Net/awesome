# Awesome

A reference hub for discovering, comparing, and annotating high-quality resources about:

- AI foundations
- AI agents
- .NET + AI
- Benchmarks, evals, and tooling

This repo is not intended to host primary implementations. Its purpose is to curate strong references, keep them well categorized, and maintain a consistent format for fast lookup and comparison.

## How To Add A New Link

1. Identify which category the resource belongs to.
2. Open the matching file under `lists/`.
3. Add a new entry using the template in [templates/entry-template.md](./templates/entry-template.md).
4. If the resource does not fit any existing file, review [docs/tagging-guide.md](./docs/tagging-guide.md) before creating a new category.

## Quick Rules

- Each resource should have one primary home.
- If a resource relates to multiple areas, place it in the best-fit location and use `Compare with` to point to adjacent references.
- Keep `Why it matters` short, clear, and comparison-friendly.
- Prefer official repos, official docs, papers, benchmarks, and strong technical writeups.
